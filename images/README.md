## Required Files

| Filename | Description | Recommended size |
|----------|-------------|------------------|
| `hackersim-1.png` | HACKERSIM logo / branding image | 1280x720 (16:9) |
| `hackersim-2.png` | Desktop view with all applications | 1280x720 (16:9) |

## Tips

- Format: PNG or JPG (PNG preferred for UI screenshots)
- Aspect ratio: 16:9 looks best in the slider (320px height container)
- Compression: Keep under 300KB each for fast loading
- Naming: Lowercase, exact filename match required
